DLScanner manual test harness

Serve, don't open from disk. Run python3 -m http.server in the plugin root and open http://localhost:8000/test/harness.html so this page can fetch() the live JavaScript out of DLScanner.pm (single source of truth — no copy to drift).

About "replay": synthetic KeyboardEvents are untrusted, so the browser will not fire its native Ctrl+J / Ctrl+6 shortcuts for them — replay here verifies field population in a real browser + jQuery and that the handler calls preventDefault(). Proving Downloads/tab-switch no longer happens requires a real scanner (trusted events) against KTD — that's the final confirmation step.
